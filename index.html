Java Path Tracer

University assignment: Course CG Project

Java - Path Tracing - Graphics

University course during the second semester under Prof. dr. ir. Dutré. The aim of this course was to build a path tracer pretty much from scratch and perform several scientific tests on the convergence speed for different algorithms and parameters. The feature-set is rather limited but it is able to render Wavefront obj objects lit by point, area or environment lighting with NEE. Diffuse and (modified) phong BRDFs can be evaluated and importance sampled. More BRDFs like Cook-Torrance and Ward-Duer where implemented but not importance sampled. The entire scene uses a basic BVH, build bottom-up, as acceleration structure for efficient ray intersection and any-hit queries.

The rendering execution as well as the BVH building is fully multi-threaded on the CPU.

Unfinished business
This course was only a one semester and worth 3 studypoints, so the features are rather limited. There where a lot of aspects and algorithms I would have loved to have implemented like:
  • Progressive Multi-Jittered Sample Sequences
  • GGX BSDF with importance sampling
  • True Multiple-Importance sampling
  • ...
I was able to implement a basic version of environment importance sampling using binning based on HDR's luminance values. Altough this method is relatively basic, it yielded much improved convergence speed for scenes lit using an environment map.
